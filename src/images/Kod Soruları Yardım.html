Adım 2: React Bileşeninde Kullanım

className özelliği ile istediğiniz ikonu kolayca ekleyebilirsiniz:

javascript
import React from 'react';

const Leftpanel = () => {

      </div>

export default Leftpanel;

Not:

  • Pro Sürümü Gerekmez: İstediğiniz fa-pen ikonu ücretsiz solid paketinde yer alır.

Adım 2: İçerikleri Kapsayan Yapının Genişliğini Ayarlayın

Eğer sol panelin dışında başka içerikler de varsa ve Grid yapısını kullanıyorsanız, xs prop'unu belirleyerek panellerin genişliklerini kontrol edebilirsiniz.

Örneğin:

javascript
<Grid container>
  <Grid item xs={2}> {/* Sol panelin genişliği */}
    <Leftpanel />
  </Grid>
  <Grid item xs={10}> {/* Sağdaki içerik */}
    <RightContent />
  </Grid>
</Grid>

Burada xs={2} ile sol panelin genişliğini ayarlıyoruz. Bu, sayfanın %16.67’sini (12 kolon üzerinden) kaplar. xs={10} sağdaki içerik için alan bırakır.
